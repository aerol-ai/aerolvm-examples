Chrome setup

  1. Set a SOCKS5 proxy in Chrome to the host and port shown above. If you already use a proxy manager, set scheme socks5.
  2. For a dedicated proxied Chrome on macOS, copy the launch command below and run it in Terminal.
  3. Chrome does not support SOCKSv5 authentication, so this demo intentionally runs the SOCKS endpoint without auth.
  4. After you are done, close the proxied Chrome window or clear the proxy settings and then destroy the sandbox.
