Chrome setup

  1. Open chrome://extensions and enable Developer mode.
  2. Load the unpacked extension from the local customer-facing/burner-vpn/chrome-extension folder.
  3. Paste the Dashboard URL, Username, and Password shown here into the popup.
  4. Click Attach Current Tab. The extension reloads the tab and keeps your existing Chrome cookies/session.
